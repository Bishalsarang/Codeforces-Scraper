
# Codeforce-Contests-Scrapper
A simple scrapper made in python for the purpose of downloading codeforce contests as pdf


Requirements:
1. wkhtmltopdf[https://wkhtmltopdf.org/]
2. pdfkit
3. requests

Usage:
```python scrapper.py```
